---

## 📖 Overview

Nidaa is a modern web platform developed to improve the organization and delivery of humanitarian assistance. It provides a centralized system where users can submit support requests, administrators can review and manage applications, and communities can participate in meaningful aid initiatives.

The platform is designed with scalability, usability, and real-world humanitarian impact in mind.

---

## ✨ Key Features

### 👤 User Module
- Secure registration and authentication
- Personal dashboard
- Submit humanitarian aid requests
- Manage account settings
- Access community resources

### 🛡️ Administration Module
- Administrative control panel
- User management system
- Review incoming requests
- Approve / reject applications
- Monitor platform activity

### ⚙️ System Features
- Responsive user interface
- Structured MVC architecture
- Spring Boot backend services
- Database integration
- Maintainable and scalable codebase
